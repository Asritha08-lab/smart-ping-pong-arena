# Smart Ping Pong Arena

Smart Ping Pong Arena is an interactive browser-based Ping Pong game developed using HTML5 Canvas, JavaScript, and CSS. The game offers both Single Player (AI) and Two Player modes with real-time gameplay, collision detection, sound effects, countdown timers, fullscreen support, and a modern gaming interface.

## Live Demo

https://asritha08-lab.github.io/smart-ping-pong-arena/

## Features

- Single Player Mode with AI Opponent
- Two Player Mode
- Multiple Difficulty Levels
- Real-Time Collision Detection
- Dynamic Ball Physics
- Countdown Before Game Start
- Pause and Resume Functionality
- Winner Screen
- Restart Game Option
- Main Menu Navigation
- Fullscreen Support
- Sound Effects
- Responsive User Interface
- HTML5 Canvas Rendering

## Controls

### Player 1

| Action | Key |
|----------|----------|
| Move Up | W |
| Move Down | S |

### Player 2

| Action | Key |
|----------|----------|
| Move Up | ↑ |
| Move Down | ↓ |

### Game Controls

| Action | Key |
|----------|----------|
| Pause | P |
| Resume | R |
| Main Menu | ESC |

## Gameplay Features

### Single Player Mode

Play against an AI-controlled paddle with adjustable difficulty levels:

- Easy
- Medium
- Hard

### Two Player Mode

Compete locally with another player using separate keyboard controls.

### Countdown System

A countdown timer is displayed before the match begins:

```text
3
2
1
GO!
```

### Winner Screen

When a player reaches the winning score, a dedicated winner screen is displayed with options to:

- Play Again
- Return to Main Menu
